# Remote PC Control via HTTP (Boost.Asio / Beast)

Простое фоновое приложение на C++ для удаленного мониторинга и базового управления состоянием персонального компьютера под управлением Windows с мобильного телефона.

## Архитектура проекта
Проект состоит из двух частей:
1. **Клиентское приложение (данный репозиторий)** — фоновый HTTP-сервер, устанавливаемый на управляемый ПК.
2. **Сервер-регистратор** — отдельная программа, которая автоматически определяет и сохраняет IP-адрес запущенного клиента при его подключении к сети. 
   * Ссылка на сервер-регистратор: [ВСТАВЬТЕ_ССЫЛКУ_СЮДА]

## Полный функционал
Приложение работает в фоновом режиме и выполняет следующие команды, отправленные через HTTP/POST/GET запросы:

* **Автозагрузка**: При первом запуске автоматически и безопасно (с поддержкой путей с пробелами) прописывается в реестр текущего пользователя (`HKCU`), обеспечивая старт вместе с Windows.
* **Скрытый режим**: Скомпилировано через `WinMain` и работает без отображения консольного окна.
* **Автоматическое уведомление об IP**: При старте пытается отправить пакет на сервер-регистратор, чтобы владелец всегда знал актуальный IP-адрес компьютера в сети.
* **Снятие скриншотов**: По запросу делает снимок экрана через фоновый процесс PowerShell, временно сохраняет его на диск, передает картинку на телефон в формате PNG и мгновенно удаляет её без блокировки файлов.
* **Выполнение консольных команд**: Позволяет удаленно запускать любые системные утилиты и программы (например, `calc`, `notepad`) в скрытом режиме (`SW_HIDE`).
* **Дистанционное выключение**: При получении команды `shutdown` корректно завершает работу Windows без вызова критических ошибок сокета.
* **Интерактивный режим (`/hack`)**:
  * Запускает цикл из 10 открывающихся окон командной строки (`cmd.exe`).
  * Выводит системное диалоговое окно `MessageBox` с критической ошибкой.
  * Программно отключает питание монитора на 1 секунду, после чего возвращает его в рабочее состояние.

## Требования для сборки
* **ОС**: Windows 10 / 11
* **Компилятор**: MSVC (Visual Studio 2022) с поддержкой C++17 или новее.
* **Библиотеки**: [Boost](https://boost.org) (Asio, Beast, System).

## Сборка и запуск
1. Установите библиотеки Boost и укажите пути к ним в настройках вашего компилятора/IDE.
2. Скомпилируйте проект в режиме **Release / x64**.
3. Запустите полученный `.exe` файл. Приложение автоматически свернется в фон и добавится в автозагрузку.

> **Важно**: Приложение слушает локальный порт `5005` и ожидает входящие HTTP-запросы. Убедитесь, что брандмауэр Windows не блокирует данный порт.
